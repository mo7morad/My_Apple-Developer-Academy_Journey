## Overview

**Help an Academy Friend** was a team introduction to UI/UX design and design thinking using **Sketch**. The focus shifted from writing code to empathizing with the user, creating wireframes, working with artboards, and understanding the design-to-development handoff process.

> [!TIP]
> Empathy is the core of this challenge—designing an experience tailored to a friend with specific needs.

## Learning Objectives

- **Sketch basics**: Artboards, shapes, text styles, and layer organization.
- **UI/UX design**: Laying out screens with visual hierarchy and proper spacing.
- **Design thinking**: Empathizing with users and iterating on a visual idea.
- **Asset export**: Preparing images and icons for developers.
- **Design handoff**: Communicating design decisions effectively to a development peer.

## Getting started

To view the deliverables:

1. Install **Sketch** from [sketch.com](https://www.sketch.com/) or the Mac App Store.
2. Open `My individual Design.sketch`.
3. Browse the artboards in the left-hand layers panel.
